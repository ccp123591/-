# FitCoach — 居家健身动作纠正与语音陪练系统

基于浏览器端骨架识别的居家健身 PWA 工具，提供实时动作计数、角度纠正、节奏评分与语音陪练。

纯前端运行，无需服务器、无需账号，所有数据本地存储，隐私安全。

---

## 功能特性

### 核心功能

- **实时姿态检测** — 基于 MediaPipe Pose 的浏览器端骨架识别，自动捕捉关键关节角度
- **智能动作计数** — 状态机驱动的精准计数，支持深蹲与前屈伸展两种动作
- **角度纠正提示** — 实时监测动作角度，动作不到位时给出语音纠正建议
- **综合评分系统** — 节奏评分（动作频率稳定性）+ 稳定度评分（关节角度一致性）
- **语音陪练** — 中文 TTS 语音播报计数、鼓励话语、纠正提示
- **节拍器** — Web Audio API 实现的节拍器辅助节奏控制

### 数据管理

- **训练记录** — IndexedDB 持久化存储每次训练数据
- **数据可视化** — Canvas 绘制的柱状图（次数）+ 折线图（评分）
- **CSV 导出** — 支持单次报告导出和全部记录批量导出

### PWA 支持

- **离线可用** — Service Worker 缓存所有本地资源
- **桌面安装** — 支持添加到桌面作为独立应用运行

## 技术栈

| 类别 | 技术 |
|------|------|
| 姿态检测 | MediaPipe Pose (BlazePose) |
| 摄像头采集 | MediaPipe Camera Utils |
| 骨架绘制 | Canvas 2D API |
| 语音合成 | Web Speech API (SpeechSynthesis) |
| 节拍器 | Web Audio API |
| 数据存储 | IndexedDB + localStorage |
| 图表绘制 | Canvas 2D (原生实现) |
| 离线缓存 | Service Worker |

## 快速开始

### 方式一：一键启动（Windows）

双击 `启动.bat`，脚本会自动启动本地 HTTP 服务器并打开浏览器。

> 需要已安装 Python 3。

### 方式二：手动启动

在项目根目录下运行任意 HTTP 服务器：

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8080
```

打开浏览器访问 `http://localhost:8080`。

> **注意**：摄像头访问需要 HTTPS 或 localhost，直接打开 HTML 文件无法使用摄像头功能。

## 使用说明

### 训练流程

1. 选择动作类型（深蹲 / 前屈伸展）
2. 设置目标次数
3. 点击「开始训练」，允许摄像头权限
4. 面对摄像头，确保全身可见
5. 按照提示完成动作，系统自动计数和评分
6. 达到目标次数后自动结束，显示训练报告

### 设置调整

- **角度阈值** — 根据个人柔韧性调整下蹲/站立的判定角度
- **节拍 BPM** — 控制节拍器速度，同时影响节奏评分基准
- **TTS 语速** — 调整语音提示的播报速度

---

## 评分算法

### 节奏评分（50%权重）

计算每次动作间隔与目标 BPM 间隔的平均绝对偏差（MAE），偏差越小分数越高。

### 稳定度评分（50%权重）

取动作最低点附近帧的关节角度标准差，标准差越小表示动作越稳定，分数越高。

### 综合评分

`综合评分 = 节奏评分 × 0.5 + 稳定度评分 × 0.5`
